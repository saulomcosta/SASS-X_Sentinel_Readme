# 📜 Changelog

Todas as mudanças relevantes do **SASS-X Sentinel** são documentadas neste arquivo.

O objetivo deste histórico é apresentar a evolução da plataforma, destacando novas capacidades, melhorias arquiteturais e marcos importantes do produto.

Este projeto segue os princípios do **Semantic Versioning (SemVer)**:

```
MAJOR.MINOR.PATCH
```

- **MAJOR** → mudanças arquiteturais significativas ou incompatíveis.
- **MINOR** → novas funcionalidades mantendo compatibilidade.
- **PATCH** → correções, otimizações e melhorias internas.
